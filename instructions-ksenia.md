# Твой астролог в Claude Code

Привет! Я перенёс твоего астролога из ChatGPT в Claude Code. Он знает твою натальную карту (теперь ещё и в ведической системе) и помогает с прогнозами, характером, выбором времени и осмыслением того, что происходит.

Установка — на выбор, любой из двух способов.

## Способ 1 (проще всего) — через Claude Code
Открой **Claude Code** и вставь это сообщение:

> Склонируй https://github.com/rakhmondaler/fates-of-xenia и запусти install.sh — установи этот скилл.

Claude сам всё скачает и поставит. Пара минут, и готово.

## Способ 2 — одной строкой в Терминале
Открой приложение **Терминал** (Spotlight через ⌘+пробел → набери «Терминал»), вставь эту строку и нажми Enter:

```
curl -fsSL https://raw.githubusercontent.com/rakhmondaler/fates-of-xenia/main/install.sh | bash
```

Когда увидишь «✅ Готово!» — установлено.

## Как пользоваться
Открой **Claude Code** и просто напиши своими словами (или начни с команды `/fates`), например:

- «что у меня по транзитам сегодня?»
- «расскажи про мой характер по карте»
- «какой сейчас у меня период?»
- «помоги понять, что со мной сейчас происходит»
- «когда лучше начать новое дело?»

Всё. Он сам подтянет твою карту и ответит.
