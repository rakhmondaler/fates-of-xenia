# Персональный астролог Ксении — скилл для Claude Code

Воссоздание её кастомного GPT (астролог, знающий натальную карту) в виде скилла
Claude Code. Западная + ведическая системы, прогнозы по транзитам, портрет по карте,
выбор времени, осмысление событий. Данные — приватные, репозиторий приватный.

Эта инструкция — для тебя (Далера). Короткая инструкция для Ксении — в файле
`instructions-ksenia.md`.

## Что внутри

```
astro-skill/
├── SKILL.md                  ← мозг: роль, голос, этика, 3 режима, логика чтения
├── install.sh                ← установщик (перед отправкой → install.command)
├── README.md                 ← этот файл
├── instructions-ksenia.md    ← 2 шага для Ксении (копируешь ей в Telegram)
├── data/
│   ├── user.md               ← данные рождения
│   ├── natal-western.md      ← западная карта (из Notion)
│   ├── portrait-western.md   ← интегральный портрет (из Notion)
│   ├── aspects.md            ← аспекты и конфигурации (из Notion)
│   ├── house-rulers.md       ← управители домов (из Notion)
│   ├── harmonization.md      ← гармонизация планет (из Notion)
│   ├── planets-in-houses.md  ← влияние планет на дома (из Notion)
│   ├── natal-vedic.md        ← ведическая карта (D1, D9, накшатры, караки)
│   └── dasha.md              ← Вимшоттари-даши (сгенерировано скриптом)
└── scripts/
    ├── vimshottari.py        ← расчёт даш (чистый Python, без зависимостей)
    └── transits.py           ← живые транзиты на эфемеридах (pyswisseph)
```

Устанавливается в `~/.claude/skills/fates/` на машине Ксении.

## Запуск (делается один раз)

### 1. Создай приватный репозиторий
Залей в него содержимое этой папки. Например:

```
cd ~/Desktop/astro-skill
git init && git add . && git commit -m "Астролог Ксении"
gh repo create fates-of-xenia --private --source=. --push
```

Репозиторий **обязательно приватный** — внутри её натальная карта и психологический
портрет.

### 2. Сделай токен для доступа
GitHub → Settings → Developer settings → **Fine-grained tokens** → Generate new token:
- **Repository access:** Only select repositories → твой `fates-of-xenia`
- **Permissions:** Contents → **Read-only**
- Срок — на твоё усмотрение (можно длинный, токен read-only на один приватный репо).

Скопируй токен (`github_pat_…`). Даже если он утечёт — это доступ только на чтение
одного репозитория, и его можно отозвать в любой момент.

### 3. Впиши токен в установщик
В `install.sh` замени строку `REPO_URL` на:

```
REPO_URL="https://<ТОКЕН>@github.com/rakhmondaler/fates-of-xenia.git"
```

(Если git заупрямится — используй форму `https://x-access-token:<ТОКЕН>@github.com/...`)

Закоммить это **локально, но НЕ пуш в репо** — чтобы токен не попал в историю. Или
держи заполненный install.command вне репозитория.

### 4. Подготовь файл для Ксении
```
cp install.sh install.command
chmod +x install.command
```
Отправь ей `install.command` (Telegram/почта) и два шага из `instructions-ksenia.md`.

## Обновление данных

Поправил карту/портрет/промпт → закоммить и запушь в репо. Ксения один раз
перезапускает `install.command` — установщик чисто перезатрёт файлы (её `.venv` с
эфемеридами сохранится). Можно отправить ей тот же файл с просьбой запустить заново.

## Как работают транзиты и даши

- **Даши** (`data/dasha.md`) посчитаны раз и навсегда от долготы Луны — они
  фиксированы от рождения и не устаревают. Если меняешь данные рождения, перегенерируй:
  ```
  cd ~/Desktop/astro-skill && python3 scripts/vimshottari.py > data/dasha.md
  ```
  Сейчас актуально: Махадаша Раху (фев 2018 – фев 2036), Антардаша Раху-Меркурий
  (янв 2026 – авг 2028). Можно сверить на бесплатном Drik Panchang / Prokerala.
- **Транзиты** считаются на лету скриптом `transits.py` на эфемеридах (Swiss
  Ephemeris, аянамша Lahiri — та же, что в натале). Если эфемериды не встали, скилл
  берёт положения из веба. Скрипт выдаёт обе системы сразу.

## Что внутри ведического слоя, а чего нет

**Есть:** D1 (раши), D9 (навамша), накшатры с падами, караки (Джаймини), функциональная
и натуральная природа планет (FB/NB), шадбала, сожжение Марса, варготтама, Вимшоттари-даши.

**Сознательно не взято** (не было полной ясности по значению — твой принцип «только
до конца известное»):
- Аштакаварга (столбец Bindu) — балльная система для тайминга. Если захочешь её для
  режима «выбор времени» — пришли легенду этих столбцов с сайта, добавлю корректно.
- Коды столбца Position (MB, HM, S, PB) — не расшифрованы. V (варготтама) и
  предположительно PN (Pushkara Navamsa) — взяты.

## Заметки
- **Обращение.** Скилл говорит Ксении «ты» (тёплый личный регистр). Если она
  предпочитает «вы» — поменяй одну строку в `data/user.md` и в `SKILL.md`.
- **Голос.** За основу взят системный промпт её GPT (тёплый, поддерживающий,
  психологический) + патч на развёрнутость ответов. Плюс правило говорить живым
  языком без штампов ИИ.
- **Аянамша** вычислена из данных (тропик минус сидерика = 23°45′ = Lahiri), а не
  задана наугад.
- **Источник данных.** Западный слой — твои выверенные markdown-файлы из Notion
  (перенесены один в один; в портрете убрана одна противоречивая строка про стихии).
  Ведический слой — из присланного физического гороскопа. Даши — вычислены.
